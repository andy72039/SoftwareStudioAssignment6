## Explanation of the Design

Example:
### Operation
+ 按下 Add all或鍵盤的上的"A"，可以把當集所有角色加入圈圈中
+ 按下Clear all按紐，或鍵盤上的'C'，可將所有在圈圈中的角色清除，回到原本的位置
+ Hovering on the character: the name of the character will be revealed.
+ 可利用滑鼠拖移角色，將它加到圈圈上
+ 按下鍵盤上的'1'~'7'按鍵，可切換集數1~7
+ ...etc.

### Visualization
+ 有背景音樂
+ 利用連結線條的粗細來表示兩個角色之間的相關程度
